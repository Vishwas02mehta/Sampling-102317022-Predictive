# Sampling-102317022-Predictive
# Sampling Assignment - ML on Imbalanced Dataset

## Objective
To analyze the impact of different sampling techniques on multiple ML models using an imbalanced credit card dataset.

## Steps Performed
1. Dataset loading
2. Data balancing
3. Sample generation
4. 5 Sampling Techniques
5. 5 ML Models
6. Accuracy evaluation
7. Best sampling-model mapping

## Sampling Techniques
- RandomUnderSampler
- SMOTE
- TomekLinks
- SMOTETomek
- BalancedRandomForest

## Models Used
- Logistic Regression
- Random Forest
- Gradient Boosting
- SVM
- Naive Bayes

## Conclusion
Sampling significantly affects ML performance. Different models respond differently to each sampling method.
